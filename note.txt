1. Inisialisasi Proyek: 
    -Membuat package.json dengan npm init -y.
2. Instalasi Dependensi:
    - express 
    - mongoose 
    - dotenv (dependencies)
    - nodemon (devDependencies)
3. Konfigurasi Skrip: Menambahkan "dev": "nodemon backend/server.js" di package.json.
4. Membuat Server: 
    - Membuat server dasar dengan Express di server.js.
    - Menambahkan rute GET / yang merespons 'Server is ready!'.
5. Menjalankan Server: 
    Menggunakan npm run dev.
6. Membuat .gitignore: 
    Mengecualikan node_modules dan .env.
